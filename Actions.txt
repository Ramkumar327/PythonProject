 
🔹 1. Page Navigation Actions
 
✅ page.goto()
 
Opens a URL.
 
page.goto("https://example.com")
 
👉 Waits for page load automatically.
 
 
---
 
✅ page.reload()
 
Reloads the page.
 
page.reload()
 
 
---
 
✅ page.go_back() / page.go_forward()
 
Browser navigation.
 
page.go_back()
page.go_forward()
 
 
---
 
🔹 2. Input Field Actions
 
These are very important in real-time automation.
 
 
---
 
✅ locator.fill()
 
Clears existing text and fills new value.
 
page.locator("#username").fill("ramkumar")
 
👉 Best for forms
👉 Replaces entire text
 
 
---
 
✅ locator.type()
 
Types like a real user (character by character).
 
page.locator("#username").type("ramkumar")
 
👉 Useful for testing typing behavior
👉 Slower than fill()
 
 
---
 
✅ locator.press()
 
Press keyboard keys.
 
page.locator("#username").press("Enter")
 
👉 Can use keys like:
 
"Enter"
 
"Tab"
 
"ArrowDown"
 
 
 
---
 
🔥 Difference (Important Interview Point)
 
Method Behavior
 
fill() Clears and enters text instantly
type() Types slowly like user
press() Keyboard key action
 
 
 
---
 
🔹 3. Click Actions
 
 
---
 
✅ locator.click()
 
Clicks on element.
 
page.locator("button#login").click()
 
👉 Auto-waits for:
 
element visible
 
element enabled
 
 
 
---
 
Advanced options
 
locator.click(button="right")   # Right click
locator.click(click_count=2)    # Double click
 
 
---
 
✅ locator.dblclick()
 
Double click.
 
locator.dblclick()
 
 
---
 
✅ locator.check() / uncheck()
 
For checkboxes.
 
page.locator("#agree").check()
page.locator("#agree").uncheck()

🔹 9. Wait Actions (Very Important)
 
Playwright is auto-waiting, but sometimes manual wait needed.
 
 
---
 
✅ locator.wait_for()
 
locator.wait_for(state="visible")
 
 
---
 
✅ page.wait_for_timeout() (not recommended usually)
 
page.wait_for_timeout(3000)
 
👉 Avoid in real projects (bad practice)

🔹 Real-Time Example (End-to-End)
 
from playwright.sync_api import sync_playwright
 
with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
 
    page.goto("https://example.com/login")
 
    page.locator("#username").fill("admin")
    page.locator("#password").fill("password")
 
    page.locator("#login").click()
 
    page.locator("#dashboard").wait_for()
 
    page.screenshot(path="dashboard.png")
 
    browser.close()
 
 
---
 
🔥 Key Takeaways (Interview Ready)
 
Playwright actions are auto-waiting
 
Prefer locator-based actions (not page.click)
 
fill() is faster than type()
 
Avoid wait_for_timeout()
 
Handles frames, popups, downloads easily
