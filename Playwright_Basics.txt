🚀 1. What is Playwright (quick understanding)
 
Playwright is a modern automation framework developed by Microsoft to test web applications.
 
👉 It supports:
 
Chromium (Chrome)
 
Firefox
 
WebKit (Safari)
 
 
👉 Why it's powerful:
 
Auto wait (no need for many sleeps)
 
Fast execution
 
Supports multiple browsers
 
API + UI testing

🔹 Step 3: Your First Script
 
from playwright.sync_api import sync_playwright
 
with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
    page.goto("https://example.com")
    print(page.title())
    browser.close()
 
👉 This is your base structure.

🔹 Step 6: Pytest Integration (MOST IMPORTANT)
 
pytest is used with Playwright.
 
Install:
 
pip install pytest
 
Sample test:
 
def test_google(page):
    page.goto("https://google.com")
    assert "Google" in page.title()
 
Run:
 
pytest
 
 
---
 
🔹 Step 7: Page Object Model (POM)
 
Structure your framework like:
 
project/
  pages/
  tests/
  utils/
 
Example:
 
class LoginPage:
    def __init__(self, page):
        self.page = page
 
    def login(self):
        self.page.fill("#user", "admin")
        self.page.fill("#pass", "1234")
        self.page.click("text=Login")

response = page.request.get("https://api.example.com")
